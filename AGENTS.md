# AGENTS Instructions

## Python Execution

- In this repository, always run Python using `backend/.venv/bin/python`.
- Do not use system `python` or `python3`.
- Prefer module execution via venv Python, for example:
  - `backend/.venv/bin/python -m py_compile ...`
  - `backend/.venv/bin/python -m pytest ...`
  - `backend/.venv/bin/python -m pip ...`
